# rds-web

The application to get RDS into the new OC Web interface.

## Project setup
```
npm install
pipenv install
```

### Compiles and hot-reloads for development
```
npm run serve
pipenv run python websocket.py
```

### Compiles and minifies for production
```
docker build
```

### Lints and fixes files
```
npm run lint
```
